# RabbitMQ
## Set UP
	Download Rabbit MQ (Binary Version i.e dump)
	Install ERLang
	Set the ERLANG_HOME
	goto rabbit sbin directory and type rabbitmq-plugins enable rabbitmq_management
	goto rabbit sbin rabbitmq-server
	default user: guest
	default password: guest
	http://localhost:15672/
